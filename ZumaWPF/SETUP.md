# Инструкция по запуску проекта

## Предварительные требования

1. Установите .NET 9.0 SDK или выше
2. Установите Visual Studio 2022 или Rider (или другой IDE с поддержкой WPF)

## Первый запуск

1. Откройте решение `ZumaWPF.sln` в вашем IDE
2. Восстановите зависимости (если требуется): `dotnet restore`
3. Соберите проект: `dotnet build` или используйте команду Build в IDE
4. Запустите проект: F5 или `dotnet run`

## Структура ресурсов

При первом запуске могут быть созданы следующие файлы:

- `config.xml` - конфигурационный файл игры (создается автоматически)
- `users.xml` - файл с пользователями (создается автоматически)
- `highscores.xml` - таблица рекордов (создается автоматически)
- `savegame.xml` - файл сохранения игры (создается автоматически при сохранении)

### Опциональные ресурсы

Для полного функционала можно добавить в папку `Resources`:

```
Resources/
├── Music/
│   └── background.wav    (фоновая музыка)
├── Sounds/
│   ├── shoot.wav         (звук выстрела)
│   └── destroy.wav       (звук уничтожения шариков)
└── Images/
    ├── Levels/
    │   ├── level1.jpg    (фон уровня 1)
    │   ├── level2.jpg    (фон уровня 2)
    │   └── level3.jpg    (фон уровня 3)
    ├── Balls/            (текстуры шариков)
    └── Shooter/          (текстуры стрелка)
```

**Примечание:** Если ресурсы отсутствуют, игра будет работать, но без звуков и с базовой графикой.

## Настройка конфигурации

Отредактируйте файл `config.xml` для изменения параметров игры:

- `BallSpeed` - базовая скорость движения шариков
- `PointsPerBall` - очки за один шарик
- `ComboMultiplier` - множитель для комбо
- `BallRadius` - радиус шариков
- `MinComboSize` - минимальный размер комбинации для уничтожения
- `BallColors` - доступные цвета шариков

## Возможные проблемы

1. **Ошибка компиляции:** Убедитесь, что установлена правильная версия .NET SDK
2. **Файлы не найдены:** Файлы конфигурации и сохранений создаются автоматически при первом запуске
3. **Звук не работает:** Проверьте наличие файлов в папке `Resources/Music` и `Resources/Sounds`

## Управление

- **Мышь** - наведение и стрельба
- **ESC** - пауза/возобновление игры
- **Левая кнопка мыши** - выстрел шариком

## Тестирование

1. Запустите игру
2. Введите имя пользователя
3. Выберите "Новая игра" или "Выбор уровня"
4. Играйте!
